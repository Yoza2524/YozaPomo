## ✨ 特性

- 🍅**番茄工作法：**自定义专注时间，合理安排自己的注意力。
- 🎉**超时专注**：专注时间结束提醒，超时后间隔时间提醒，不再忍受被强行打断的痛苦。
- 🪜**TODO List 和 番茄钟 的结合：**双击TODO进入专注模式，无感的记录工作内容和时长。
- ✍️**便捷的记录：**专注模式结束、TODO任务做完后弹出总结框，轻松记录、无感总结。
- 📶**专注检测提醒：**在一定时间内会检测输入，一段时间内无输入自动结束专注并标记异常，减小时长统计误差。
- 📄**Markdown支持：**记录、存储和展示使用Markdown，条目清晰、便于迁移。
- 💻**跨平台支持：**未来一定会支持的… … 吧。
