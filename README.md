## ✨ 特性

- 🍅**番茄工作法：**在专注时间结束后，使用合适的音量和动画提示休息，永远不会强制打断你。
- 🪜**TODO List 和 番茄钟 的结合：**双击TODO进入专注模式，无感的记录工作内容和时长。
- ✍️**便捷的记录：**专注模式结束、TODO任务做完后弹出总结框，轻松记录、无感总结。
- 📶**精准的时长：**在一定时间内会检测输入，一段时间内无输入自动结束专注并标记异常，减小时长统计误差。
- 📄**Markdown支持：**记录、存储和展示使用Markdown，条目清晰、便于迁移。
