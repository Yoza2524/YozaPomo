# 柚子钟(**YozaPomo**)

​		起因是我发现现在的我很难集中注意力，就在网上搜索各种的集中注意力的方案，诸如类似舒尔特方格的训练法 或是 番茄钟之类的外部工具。

​		训练的话需要长时间的训练，番茄钟这种可以快速的让我暗示自己进入状态。但又苦于市面上的番茄钟都不太好用，或是提醒太强打断了我正在做的任务，或是提醒太弱让我经常想去看看还有多久，好看的界面也是几乎没有，想在windows和mac上有差不多的体验更是天方夜谭。也就有了我想做一个随心所欲的钟的想法… …。

​		一年时间过去(这居然是记叙文啊喂)，我的Rust和Tauri的水平已经是完成了从0到0.00…01的蜕变，再加上vibe coding已经比较成熟(说不定是主要原因😋)，所以我的个人番茄钟计划现在正式启动！！！

​		啥是番茄钟？  据说很久很久以前有一个外国人觉得自己学习效率低、注意力不集中，想试试自己能不能集中注意力xx分钟，就找到了一个番茄形状的计时器，设定了25分钟工作 5分钟休息的工作方法。

​		那为什么是柚子钟呢？因为我叫yoza(肥肠合理)。

### 开始幻想

​		上面居然全是废话吗，可恶。

​		关于我对柚子钟的期望，我希望不管我在用哪台电脑，它都是出现在我显示器的右上角，上面有几个小标签记录我今天的全部任务(简单的TODO List)，下面有一个“我要专注“的按钮，按一下进入我设置市场的计时。

​		在计时结束后有悬浮在显示器正上方播放一小段动画 同时给一段简单的提示音，然后右上角弹出按钮 在我真正完成后我再去点，软件帮我记录我真正的工作时间。
