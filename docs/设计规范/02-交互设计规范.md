# 交互设计规范

> 本文档定义 YozaPomo 所有用户交互行为的标准，包括鼠标操作、键盘快捷键和状态转换。

---

## 1. 悬浮窗交互

### 1.1 基础交互

| 操作 | 行为 | 说明 |
|------|------|------|
| 左键拖拽 | 移动窗口 | 在标题栏区域拖拽 |
| 右键点击 | 弹出菜单 | 显示"显示管理界面"、"退出" |
| 双击标题栏 | 最大化/还原 | 暂不支持，保持当前位置 |
| 悬停窗口 | 显示边框 | 从透明变为轻微可见边框 |

### 1.2 TODO 标签交互

```
状态转换图：

  [默认] ──双击──> [激活/专注中]
    ↑                   │
    │                   │ 双击
    │                   ↓
    │              [暂停]
    │                   │
    │                   │ 右键 → 结束
    │                   ↓
    └──── 右键结束 ──── [完成]
```

| 操作 | 默认状态 | 激活状态 | 暂停状态 |
|------|----------|----------|----------|
| 单击 | 无操作 | 无操作 | 无操作 |
| 双击 | 激活并开始专注 | 暂停专注 | 恢复专注 |
| 右键 | 弹出菜单（开始/结束） | 弹出菜单（暂停/结束） | 弹出菜单（恢复/结束） |

**右键菜单项**：
- 默认状态：「开始专注」「结束 TODO」
- 激活状态：「暂停专注」「结束 TODO」
- 暂停状态：「恢复专注」「结束 TODO」

**结束 TODO 流程**：
1. 弹出 Markdown 笔记窗口（可选填写）
2. 保存笔记
3. 标记 TODO 为完成
4. 从悬浮窗移除

### 1.3 专注按钮交互

```
状态转换图：

  [空闲] ──点击──> [专注中]
    ↑                   │
    │                   │ 倒计时归零
    │                   ↓
    │              [倒计时结束]
    │                   │
    │    ┌──────────────┼──────────────┐
    │    ↓              ↓              ↓
    │ [开始休息]   [结束专注]    [我还在专注]
    │    │              │              │
    │    └──────────────┴──────────────┘
    │                   │
    └───────────────────┘
```

| 按钮 | 出现时机 | 行为 |
|------|----------|------|
| 开始专注 | 空闲状态 | 开始倒计时 |
| 暂停 | 专注中 | 暂停倒计时 |
| 恢复 | 暂停状态 | 恢复倒计时 |
| 开始休息 | 倒计时归零后 | 开始休息倒计时 |
| 结束专注 | 倒计时归零后 | 直接结束专注 |
| 我还在专注 | 倒计时归零后 | 关闭提醒，继续专注 |

### 1.4 5 分钟提醒交互

```
触发条件：专注开始后 5 分钟

行为流程：
1. 播放较小的提示音
2. 显示小型粒子动画（顶部）
3. 显示「我还在专注」按钮
4. 按钮在 1 分钟后自动消失
5. 若未点击，标记为异常专注并结束
```

### 1.5 倒计时归零交互

```
触发条件：倒计时从 >0 变为 0

行为流程：
1. 播放提示音
2. 显示粒子动画（较大）
3. 按钮区域变为：
   ┌──────────────────┐
   │  开始休息  25:00  │  ← 休息按钮 + 本次专注总时长
   └──────────────────┘
4. 超时后显示特殊格式：
   ⏱ 超时 02:30       ← 红色/琥珀色文字
```

## 2. 管理界面交互

### 2.1 打开/关闭

| 操作 | 行为 |
|------|------|
| 系统托盘左键 | 显示管理界面（如已隐藏） |
| 系统托盘右键 → 显示 | 显示管理界面 |
| 管理界面关闭按钮 | 隐藏窗口（不退出应用） |
| 系统托盘右键 → 退出 | 退出应用 |

### 2.2 TODO 管理页面

**页面切换**：
- 顶部 Tab 切换：「今日 TODO」「昨日未完成」「TODO 列表」
- 使用 Naive UI 的 `Tabs` 组件

**TODO 操作**：

| 操作 | 入口 | 行为 |
|------|------|------|
| 新建 | 页面顶部 "+" 按钮 | 弹出输入框，填写标题 |
| 编辑 | 双击 TODO 项 | 进入编辑模式 |
| 删除 | 右键菜单 → 删除 | 二次确认后删除 |
| 完成 | 右键菜单 → 完成 | 标记为完成 |
| 查看详情 | 单击 TODO 项 | 展开详情面板（专注历史、笔记） |

**TODO 详情面板**：
```
┌─────────────────────────────────────┐
│  📝 写周报                     ✏️ 🗑️ │
├─────────────────────────────────────┤
│  日期：2026-06-04                   │
│  专注次数：3 次                     │
│  总专注时长：1h 15min              │
├─────────────────────────────────────┤
│  专注历史                           │
│  ├─ 14:30 - 14:55 (25min) ✅       │
│  ├─ 15:00 - 15:25 (25min) ✅       │
│  └─ 15:30 - 15:40 (10min) ⚠️ 中断  │
├─────────────────────────────────────┤
│  笔记（Markdown）                   │
│  ┌─────────────────────────────────┐│
│  │ 已完成初稿，需要补充数据部分    ││
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
```

### 2.3 专注历史页面

**功能**：
- 按日期分组显示所有专注记录
- 支持日期范围筛选
- 显示每日专注统计

**列表项信息**：
```
┌─────────────────────────────────────┐
│  📅 2026-06-04                      │
│  ├─ 09:00 - 09:25 | 25min | 写周报 │
│  ├─ 10:00 - 10:25 | 25min | 复习   │
│  └─ 14:00 - 14:30 | 30min | —      │
│  合计：1h 20min | 3 次             │
└─────────────────────────────────────┘
```

### 2.4 设置页面

**设置项**：

| 设置项 | 控件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 默认专注时长 | 数字输入框 | 25 分钟 | 范围：5-120 分钟 |
| TODO 显示数量 | 数字输入框 | 3 | 范围：1-10 |
| 显示倒计时 | 开关 | 开 | 悬浮窗是否显示倒计时 |
| 通知音效 | 文件选择器 | default | 支持 mp3/wav |

## 3. 键盘快捷键

### 3.1 全局快捷键

| 快捷键 | 功能 | 作用范围 |
|--------|------|----------|
| `Ctrl+Shift+F` | 显示/隐藏悬浮窗 | 全局 |
| `Ctrl+Shift+M` | 显示管理界面 | 全局 |

### 3.3 管理界面内快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建 TODO |

| `Delete` | 删除选中项 |
| `Escape` | 关闭弹窗 |

## 4. 状态管理规则

### 4.1 专注状态

```typescript
type FocusStatus = 'idle' | 'focusing' | 'paused' | 'overtime' | 'resting'

// 状态转换规则
const transitions: Record<FocusStatus, FocusStatus[]> = {
  idle:      ['focusing'],
  focusing:  ['paused', 'overtime', 'idle'],
  paused:    ['focusing', 'idle'],
  overtime:  ['resting', 'idle'],
  resting:   ['idle'],
}
```

### 4.2 TODO 状态

```typescript
type TodoStatus = 'active' | 'focusing' | 'completed'

// 状态转换规则
const todoTransitions: Record<TodoStatus, TodoStatus[]> = {
  active:    ['focusing', 'completed'],
  focusing:  ['active', 'completed'],
  completed: [], // 终态
}
```

### 4.3 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 专注中应用崩溃 | 重启后恢复上次专注状态（从数据库） |
| 数据库操作失败 | 显示错误提示，不崩溃 |
| 窗口位置超出屏幕 | 自动调整到屏幕内 |
| 音频播放失败 | 静默失败，不影响专注 |
| 系统休眠唤醒 | 重新计算倒计时剩余时间 |

## 5. 通知与提示

### 5.1 提示音使用场景

| 场景 | 音效 | 音量 |
|------|------|------|
| 专注结束 | focus-end.mp3 | 70% |
| 5分钟提醒 | reminder.mp3 | 40% |
| 休息结束 | rest-end.mp3 | 70% |
| TODO 完成 | 无 | — |
| 错误提示 | 系统提示音 | 50% |

### 5.2 视觉提示

| 场景 | 提示方式 |
|------|----------|
| 专注结束 | 粒子动画 + 按钮变化 |
| 5分钟提醒 | 小型动画 + 按钮出现 |
| 超时 | 文字颜色变化（琥珀色） |
| TODO 完成 | 标签淡出 + 移除 |
| 错误 | Naive UI Message 组件 |
