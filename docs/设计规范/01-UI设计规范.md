# UI 设计规范

> 本文档定义 YozaPomo 的视觉设计标准，包括颜色、排版、间距、组件样式等。

---

## 1. 设计原则

1. **极简**：去除一切不必要的元素，减少视觉干扰
2. **透明**：悬浮窗与桌面融合，不遮挡工作区
3. **苹果风格**：圆角、毛玻璃、柔和阴影
4. **专注**：UI 为专注服务，不抢注意力

## 2. 色彩系统

### 2.1 主色

| 用途 | 颜色名 | 色值 | 使用场景 |
|------|--------|------|----------|
| 主色 | Indigo | `#6366f1` | 按钮、链接、强调 |
| 主色浅 | Indigo Light | `#818cf8` | 悬停状态 |
| 主色深 | Indigo Dark | `#4f46e5` | 按下状态 |

### 2.2 状态色

| 用途 | 颜色名 | 色值 | 使用场景 |
|------|--------|------|----------|
| 成功/专注中 | Green | `#22c55e` | 专注中状态、完成标记 |
| 警告/超时 | Amber | `#f59e0b` | 超时显示 |
| 错误/异常 | Red | `#ef4444` | 异常专注、删除 |
| 信息 | Blue | `#3b82f6` | 提示信息 |

### 2.3 中性色

| 用途 | 色值 | 使用场景 |
|------|------|----------|
| 文字主色 | `#1f2937` | 标题、正文 |
| 文字次色 | `#6b7280` | 辅助文字、时间 |
| 文字禁用 | `#9ca3af` | 禁用状态文字 |
| 边框 | `#e5e7eb` | 分割线、边框 |
| 背景 | `#f9fafb` | 页面背景 |
| 白色 | `#ffffff` | 卡片、弹窗背景 |

### 2.4 悬浮窗专用

```css
/* 悬浮窗背景：半透明毛玻璃 */
.floating-bg {
  background: rgba(255, 255, 255, 0.85);
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
}

/* TODO 标签激活状态呼吸灯 */
.todo-active-glow {
  box-shadow: 0 0 15px rgba(34, 197, 94, 0.3);
  animation: breathe 2s ease-in-out infinite;
}
```

## 8. 动画规范

### 8.1 过渡动画

```css
/* 所有可交互元素使用统一过渡 */
.transition-base {
  transition: all 0.2s ease-in-out;
}

/* 呼吸灯动画（TODO 激活状态） */
@keyframes breathe {
  0%, 100% {
    box-shadow: 0 0 0 0 rgba(34, 197, 94, 0.4);
  }
  50% {
    box-shadow: 0 0 0 8px rgba(34, 197, 94, 0);
  }
}

/* 进入动画 */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### 8.2 粒子动画规范

| 属性 | 值 | 说明 |
|------|-----|------|
| 粒子数量 | 30-50 | 专注结束时 |
| 粒子大小 | 2-6px | 随机 |
| 粒子颜色 | 主色系 | 使用渐变 |
| 动画时长 | 2-3s | 从中心向外扩散 |
| 缓动函数 | `ease-out` | 自然减速 |

### 8.3 窗口动画

- 悬浮窗显示/隐藏：`opacity` 渐变 + `transform: scale(0.95→1)`
- 管理界面打开：从系统托盘位置滑入
- 弹窗：`opacity` + `transform: translateY(20px→0)`

## 9. 响应式断点

由于是桌面应用，主要考虑不同屏幕尺寸：

| 断点 | 宽度 | 悬浮窗调整 |
|------|------|-----------|
| 小屏 | < 1366px | 缩小间距，字号不变 |
| 标准 | 1366-1920px | 默认样式 |
| 大屏 | > 1920px | 适当增大间距 |

## 10. 无障碍设计

1. **对比度**：文字与背景对比度 >= 4.5:1（WCAG AA）
2. **焦点指示**：所有可交互元素有明确的焦点样式
3. **键盘导航**：支持 Tab 键切换、Enter 确认、Esc 取消
4. **屏幕阅读器**：关键元素添加 `aria-label`
