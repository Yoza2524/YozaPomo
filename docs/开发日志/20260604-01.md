## 阶段一：数据库层

### 目标

实现 SQLite 数据库完整 CRUD，包括 Rust 端命令和前端 composable

### 完成内容

**Rust 端**
- 用 `rusqlite`（bundled）替代 `tauri-plugin-sql`，直接管理数据库连接
- 实现 `Database` 结构体：连接池、WAL 模式、迁移执行
- 实现 TODO CRUD 命令（6 个）：`create_todo`、`get_todos`、`get_all_todos`、`get_todo`、`update_todo`、`delete_todo`
- 实现专注会话命令（7 个）：`create_focus_session`、`get_focus_session`、`get_todo_focus_sessions`、`get_today_focus_stats`、`get_recent_focus_sessions`、`update_focus_session`、`delete_focus_session`
- 实现设置命令（4 个）：`get_all_settings`、`get_setting`、`set_setting`、`update_settings`
- 新增依赖：`rusqlite 0.31`、`chrono 0.4`
- 移除依赖：`tauri-plugin-sql`

**前端**
- 创建 TypeScript 类型定义（`types/todo.ts`、`focus.ts`、`settings.ts`）
- 实现数据库调用封装（`utils/database.ts`）
- 实现时间格式化工具（`utils/format.ts`）
- 实现组合式函数：`useTodos`、`useFocus`、`useSettings`
- 实现 Pinia Store：`todoStore`、`focusStore`、`settingsStore`
- 创建完整目录结构（components/floating/management/composables/stores/types/utils）
