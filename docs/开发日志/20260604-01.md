## 阶段三：计时器核心

### 目标

实现完整专注计时逻辑：倒计时、暂停/恢复、超时、5分钟提醒、休息

### 完成内容

**useTimer 组合式函数**
- setInterval 驱动，每秒 tick
- start/pause/resume/stop/reset 完整状态机
- 倒计时归零后自动进入超时（负数计时）
- 5 分钟后触发提醒，提醒出现 1 分钟后自动消失

**focusStore 更新**
- 集成 useTimer，暴露计时器 ref 给组件
- startFocus 创建数据库会话 + 启动计时器
- completeFocus 结束计时器 + 写入数据库
- abortFocus 异常结束（"我还在"超时）
- 监听提醒自动消失 → 触发 abortFocus

**FocusPanel.vue**
- 等宽数字倒计时显示（36px monospace）
- 超时显示 ↑MM:SS↑（琥珀色）
- 进度条（绿→黄渐变）
- 专注总时长提示

**FocusButton.vue**
- 空闲 → 开始专注 | 专注中 → 暂停 | 暂停 → 继续 | 超时 → 结束专注
- 超时后显示"开始休息"次要按钮
- 按钮样式随状态变化

**ReminderBanner.vue**
- "还在专注吗？"+"我还在"按钮
- 淡入动画
- 用户点击 dismiss 或 1 分钟后自动消失
