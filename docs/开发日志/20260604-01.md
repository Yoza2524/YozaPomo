实现完整专注计时逻辑：倒计时、暂停/恢复、超时、5分钟提醒、休息

- setInterval 驱动，每秒 tick

- 倒计时归零后自动进入超时（负数计时）
- 5 分钟后触发提醒，提醒出现 1 分钟后自动消失

- 修复：

  - 用 ResizeObserver 让悬浮窗自动收缩到有内容的区域，透明区域自然不存在

**Bug 7: 管理界面窗口变得超级矮（900×136）** ✅

- 修复：将窗口标签检测从 `onMounted` 提前到 setup 阶段同步执行，从根源阻止 useAutoResize 在管理窗口误执行
- 管理界面配置调整为 900×700，resizable: false

**Bug 8: 管理界面右键菜单未禁用** ✅
- 问题：管理界面右键会弹出浏览器默认右键菜单
- 修复：ManagementWindow.vue 根容器添加 `@contextmenu.prevent`
