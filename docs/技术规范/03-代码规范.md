# 代码规范

> 本文档定义 YozaPomo 项目的代码风格、命名约定和最佳实践标准。

---

## 1. 通用原则

1. **可读性优先**：代码是写给人看的，顺便让机器执行
2. **单一职责**：每个函数、组件、模块只做一件事
3. **最小暴露**：只暴露必要的接口，内部实现隐藏
4. **一致性**：同一文件、同一模块内保持风格统一

### 2.2 命名规范

| 类别 | 格式 | 示例 |
|------|------|------|
| 变量 | camelCase | `todoCount`, `isFocused` |
| 常量 | UPPER_SNAKE_CASE | `MAX_TODO_DISPLAY`, `FOCUS_DURATION_DEFAULT` |
| 函数 | camelCase | `formatTime()`, `createTodo()` |
| 类型/接口 | PascalCase | `Todo`, `FocusSession` |
| 枚举值 | PascalCase | `FocusStatus.Active` |
| 布尔变量 | `is`/`has`/`should` 前缀 | `isActive`, `hasNotes` |
| 事件处理 | `handle` 前缀 | `handleDoubleClick` |
| 模板事件 | `on` + 事件名 | `@on-start` |

### 2.3 TypeScript 规范

```typescript
// ✅ 正确：使用接口定义对象形状
interface Todo {
  id: string
  title: string
  date: string
  notes: string
  focusSessions: FocusSession[]
  createdAt: Date
  updatedAt: Date
}

// ✅ 正确：使用 type 定义联合类型
type FocusStatus = 'idle' | 'focusing' | 'paused' | 'completed'

// ✅ 正确：函数参数明确类型
function formatDuration(seconds: number): string {
  const mins = Math.floor(seconds / 60)
  const secs = seconds % 60
  return `${mins.toString().padStart(2, '0')}:${secs.toString().padStart(2, '0')}`
}

// ❌ 错误：使用 any
function formatDuration(seconds: any): string { ... }

// ❌ 错误：隐式类型
const count = 0  // 应该是 const count: number = 0 或让 TS 推断
```

### 2.4 Pinia Store 规范

```typescript
// stores/focusStore.ts
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import type { FocusSession, FocusStatus } from '@/types/focus'

export const useFocusStore = defineStore('focus', () => {
  // State（使用 ref）
  const status = ref<FocusStatus>('idle')
  const currentSession = ref<FocusSession | null>(null)
  const remainingSeconds = ref(0)

  // Getters（使用 computed）
  const isFocusing = computed(() => status.value === 'focusing')
  const progress = computed(() => {
    if (!currentSession.value) return 0
    return 1 - remainingSeconds.value / currentSession.value.plannedDuration
  })

  // Actions（普通函数）
  function startFocus(duration: number) {
    status.value = 'focusing'
    remainingSeconds.value = duration
  }

  function pauseFocus() {
    status.value = 'paused'
  }

  return {
    status,
    currentSession,
    remainingSeconds,
    isFocusing,
    progress,
    startFocus,
    pauseFocus,
  }
})
```

## 7. 禁止事项

| 禁止 | 原因 |
|------|------|
| 使用 `any` 类型 | 破坏类型安全 |
| 使用 `var` | 作用域不明确 |
| 使用 `!important` | 样式优先级失控 |
| 使用 `console.log` 调试 | 生产代码中应移除 |
| 硬编码魔法数字 | 使用常量替代 |
| 深层嵌套（>3 层） | 降低可读性 |
| 复制粘贴代码 | 应抽取为函数或组件 |
| 忽略 TypeScript 错误 | 必须显式处理 |
