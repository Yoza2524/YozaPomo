- 📋 需求文档 (YozaPomo)

  ### 1. 总览

  柚子钟是一个桌面效率工具，由一个**悬浮窗**和一个**管理界面**组成。悬浮窗始终位于屏幕右上角，背景透明，苹果风格圆角，显示当日TODO标签和番茄钟控件。管理界面用于管理TODO和修改设置，通过系统托盘呼出。

  ### 2. 悬浮窗功能

  - **位置与外观**：固定在屏幕右上角，透明背景，圆角，置顶。

  - **TODO Tag**：

    - 显示当天TODO（数量可在设置中限制，默认最多3个），超过设置上限在下方显示省略号。
    - 每个Tag显示标题
    - **双击**：开始任务（高亮呼吸效果）并自动进入番茄钟专注状态；如果已在专注，则绑定当前专注，并更新TODO开始时间为当前专注的开始时间（防抖）。
    - **再次双击**：暂停任务，记录时间片。
    - **右键菜单**：开始/暂停（根据当前状态显示），结束。结束任务后Tag从悬浮窗移除。
    - 有正在进行的TODO时，不可直接开始其他TODO（不能双击或右键开始）
    - TODO结束后，弹出窗口记录当前TODO备注(markdown)

  - **番茄钟区域**：

    - 默认按钮 **“开始专注”**（若无激活的TODO则独立计时；有激活TODO则绑定该TODO）。

    - 点击后进入专注，显示倒计时（可在设置设置是否显示当前内容）和 **“结束专注”** 按钮。

    - **倒计时结束当前专注不结束**，结束专注的方式包括且仅包括

      1. 点击**“开始休息”**或**“结束专注”**按钮
      2. 点击 **“我还在”** 按钮
      3. 双击或右键结束正在运行的TODO

    - 倒计时归零后

      - 在屏幕中上位置 播放动画并给出提示音
      - **“结束专注”** 按钮变为**“开始休息”** 按钮和本次总专注时长。
      - 倒计时修改为当前专注的超时时长，倒计时显示为“↓xx:xx:xx↓”，超时时长显示为“↑xx:xx:xx↑”

    - **每5分钟提醒**（专注超过5分钟触发）：在悬浮窗内播放更小动画+更轻提示音(减小音量)，动画消失位置出现 **“我还在”** 按钮，该按钮1分钟后消失。若消失前未被点击，则强制结束本次专注，不弹窗，并记录为异常专注。

      如果在按钮出现时，点击**“结束专注”** 按钮，则专注结束， **“我还在”** 按钮消失

      > 修改记录:
      >  “还在吗”按钮生效时不出现，每0.25s检测输入，3s内无输入则按钮出现，30s内无输入异常结束当次专注
      >  专注状态下每5分钟，触发一次“还在吗”按钮(无特效)
      >  超时专注状态下每3分钟，触发一次“还在吗”按钮，且伴有小烟花特效
    
    - 结束专注后，弹出窗口记录当前专注备注(markdown)。结束TODO时，只显示TODO备注窗口
  
  ### 3. 管理界面功能
  
  - **打开方式**：
  
    - 系统托盘右键菜单 → “主页面” 
  
  - **TODO管理：**
  
    TODO包含属性：TODO名称、结束时间、总记录时长(每次开始到暂停)、创建时间、专注次数、备注
  
    - **操作**：
  
      - 右上角添加
  
        - 日期(默认当天)
        - 多次专注开始、暂停（/结束）时间
        - 备注（markdown编辑）
  
      - 编辑
  
        - 日期
        - 多次专注开始、暂停（/结束）时间
        - 备注（markdown编辑）
  
      - 查看备注
  
        mark阅读，最上方为TODO的备注，下方为多个关联的关注备注
  
      - 删除
  
    - **页面**
  
      - **今日TODO**
  
      - **昨日未完成TODO**
  
      - **TODO列表**
  
  - **专注管理：**
  
    专注包含属性：专注开始时间、专注结束时间、原定专注时间、专注时间、关联TODO、专注备注
  
    - 页面
      - 专注列表
    - 操作
      - 编辑
        - 开始时间
        - 结束时间
        - 关联TODO
        - 备注

### 4. 设置界面

设置后立即生效

- **打开方式**：
  - 系统托盘右键菜单 → “主页面”  →  “上方设置Tab”
  - 系统托盘右键菜单 → “设置”
- **设置项**：
  - 默认专注时长（默认25min）
  - 显示TODO数量（默认3）
  - 是否显示倒计时数字（默认开）
  - 提示音频文件（提供默认提示音，选择mp3/wav）

### 5. 数据存储

- SQLite 通过 `tauri-plugin-sql` 管理。
- 表：`todos`，`todo_sessions`（记录每个TODO的开始/暂停/结束时间），`pomodoro_sessions`（专注记录），`settings`。

### 6. 系统托盘

- 托盘图标，右键菜单：**打开管理界面**、**退出**。
- 左键单击可以显示/隐藏管理界面。

### 7. 技术选型

- 框架：Tauri v2 (Rust)
- 前端：Vue 3 + TypeScript
- UI库：Naive UI + Tailwind CSS
- 状态管理：Pinia
- 原生菜单：Tauri Menu API
- 数据库：SQLite (`tauri-plugin-sql`)
- 音效：Howler.js 或 Web Audio
- 动画：Canvas 粒子系统，CSS动画

### 8. 跨平台与透明窗口

- 支持 Windows / macOS / Linux。
- 悬浮窗配置：`transparent: true`，`decorations: false`，`alwaysOnTop: true`。位置动态计算（屏幕宽度 - 窗口宽度，Y轴可设偏移）。
- 管理界面为普通窗口（可关闭）。
