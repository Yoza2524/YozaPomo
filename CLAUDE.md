## 项目概述

**YozaPomo** 是一款跨平台桌面番茄钟应用，核心特性：

- **悬浮窗**：始终显示在屏幕右上角，透明背景，Apple 风格圆角
- **TODO 管理**：双击标签启动专注，支持暂停/恢复/完成
- **专注计时**：25 分钟倒计时，5 分钟提醒，超时处理
- **粒子动画**：专注结束时的视觉反馈
- **管理界面**：通过系统托盘打开，管理 TODO 和专注历史

**技术栈**：Tauri v2 (Rust) + Vue 3 + TypeScript + Naive UI + Tailwind CSS + SQLite

## 关键约定

### 1. 代码风格

- Vue 组件使用 `<script setup lang="ts">`
- TypeScript 严格模式（`strict: true`）
- 使用 Composition API，不使用 Options API
- 命名规范见 `docs/技术规范/03-代码规范.md`

### 2. 文件组织

- 按功能域划分：`floating/`（悬浮窗）、`management/`（管理界面）
- 组合式函数封装可复用逻辑
- 类型定义集中在 `types/` 目录
- 详细结构见 `docs/技术规范/02-项目结构.md`

### 3. 状态管理

- 使用 Pinia，不使用 Vuex
- 每个功能域一个 Store
- Store 使用 Composition API 风格
- 详细规范见 `docs/技术规范/03-代码规范.md`

### 4. 数据库

- 使用 SQLite + `tauri-plugin-sql`
- 所有查询使用参数化查询
- 迁移脚本放在 `src-tauri/migrations/` 目录
- 详细设计见 `docs/技术规范/04-数据库设计.md`

### 5. Tauri 配置

- 悬浮窗：`transparent: true`, `decorations: false`, `alwaysOnTop: true`
- 管理界面：标准窗口，通过系统托盘打开
- 权限声明放在 `src-tauri/capabilities/default.json`
- 详细配置见 `docs/技术规范/05-Tauri配置规范.md`

## 常用命令

```bash
# 启动开发服务器
pnpm tauri dev

# 构建生产版本
pnpm tauri build

# 运行前端测试
pnpm test

# 运行 Rust 测试
cd src-tauri && cargo test

# 代码检查
pnpm lint

# 代码格式化
pnpm format
```

## 注意事项

1. **需求文档是基础**：所有开发必须符合 `docs/需求文档/20260604-01.md` 的要求
2. **规范文档优先**：遇到疑问时，先查阅对应的规范文档
3. **开发日志记录**：每天更新 `docs/开发日志/` 目录下的日志文件
4. **跨平台兼容**：开发时考虑 Windows/macOS/Linux 三平台
5. **性能优先**：悬浮窗常驻，必须保持低资源占用
